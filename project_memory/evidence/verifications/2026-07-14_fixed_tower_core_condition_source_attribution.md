# 2026-07-14 固定塔核心窗口来源诊断

## 分析问题

在不引入 FL 切面结构的前提下，检验 MT/CVT 硬 QC 共同窗口中的正塔间差异究竟主要来自：

1. 季节、时段和风速条件混杂；
2. 两塔平均垂直风、湍流强度或旋转角差异；
3. 原始垂直 CO2 协方差；
4. 坐标旋转把水平协方差投影进旋转后垂直轴；
5. 协方差计算之后已经包含在 `co2_flux` 中的修正。

## 核心窗口

- 主口径：`sector_pf`。
- 顺序筛选：全部共同窗口 `12,471`；`06:00–18:00` 后 `6,309`；再要求 MT `z/L < -0.1` 后 `3,812`；再要求 MT 风向 `180–270°` 后得到 `1,696` 个窗口、覆盖 `198` 个日期。
- 该筛选由前两批时间和条件图预先确定，不根据本轮机制变量重新调阈值。

## 条件控制方法

- 对每个机制变量单独拟合：标准化 `D ~ 标准化机制变量 + season + halfhour + MT wind-speed class`。
- `season`、精确半小时和 MT 风速等级作为加性分类控制项；一个模型只加入一个机制变量，避免协方差分量之间的强共线性使系数失去可读性。
- 不确定性使用 `3` 日循环日期块 bootstrap，`2,000` 次重复；同一天全部半小时窗口共同抽样。
- 标准化系数表示：控制季节、半小时和风速等级后，机制量增加 `1 SD` 时，`D` 改变多少个 `SD`。`partial_R2` 表示该机制量在控制项之外增加的解释比例。

## 嵌套通量分解

设 `n_air = rho_air / 0.02896546`，把 ppm 协方差换为 `umol m-2 s-1`。对每座塔：

- 原始垂直湍流项：`F_raw_w = n_air × cov(w,c)`；
- rotation 调整项：`F_rot = n_air × [a cov(u,c) + b cov(v,c) + (c-1) cov(w,c)]`；
- 后协方差修正残差：`F_corr = co2_flux - F_raw_w - F_rot`。

因此塔间差异严格闭合为：

`D = ΔF_raw_w + ΔF_rot + ΔF_corr`。

`F_corr` 汇总密度/WPL/频谱等已经进入最终 `co2_flux`、但当前主表没有独立分量字段的处理，不对其内部继续强行拆分。

平均流诊断另定义为 `F_mean = n_air × w_mean × c_mean`。它使用约 400 ppm 的背景浓度，不属于标准 EC 的 `w'c'`，也不与上述三项相加；这里只用于判断残余平均垂直风是否会产生不可接受的背景浓度输送量级。

## 预设假设与判据

- 若差异只是时间和风速混杂，控制后主要机制量的 CI 应跨零。
- 若差异主要来自真实的垂直湍流协方差，`Δcov(w,c)` 应保持强正关联，且 `ΔF_raw_w` 应占累计差异的大部分。
- 若差异主要是 rotation 产物，rotation 调整应占主要份额，并在条件模型中保持强关联。
- 若 `w_mean × c_mean` 可以作为可信局地输送，其量级应至少与 EC 差异近似可比；若高出数个数量级，则更可能反映残余坐标/流线倾斜与背景浓度相乘，不能作为局地净交换。

## 结果

核心窗口中：

- 平均 `D = 5.3482 umol m-2 s-1`；累计差异 `195.9238 gC m-2`，3 日块 95% CI `168.8631–225.7036`。
- 原始垂直湍流项：平均 `4.0497 umol m-2 s-1`，累计 `148.3536 gC m-2`，CI `122.9567–175.7229`，占点估计总差异 `75.72%`。
- rotation 调整：平均 `0.9006`，累计 `32.9935 gC m-2`，CI `25.3456–41.8206`，占 `16.84%`。
- 后协方差修正残差：平均 `0.3979`，累计 `14.5768 gC m-2`，CI `12.4567–16.8736`，占 `7.44%`。
- 最大逐窗口闭合误差 `5.33e-14 umol m-2 s-1`。

条件控制后的标准化关联：

- `Δcov(w,c)`：`beta = 0.893`，CI `0.827–0.949`，`partial_R2 = 0.789`；是唯一占主导的协方差分量。
- `Δcov(u,c)`、`Δcov(v,c)` 和 `Δrotation adjustment` 的 CI 均跨零；它们不解释核心窗口内的大部分逐窗口变化。
- `Δsigma_w`：`beta = -0.309`，CI `-0.379–-0.243`，`partial_R2 = 0.086`。因差值定义为 CVT-MT，强正 `D` 倾向出现在 MT 的 `sigma_w` 相对 CVT 更大时。
- `Δw_mean` 与 `Δtilt` 分别为 `-0.072` 和 `-0.111`，CI 不跨零但解释比例仅 `0.5%` 和 `1.3%`，属于次要条件信号。

平均流诊断为 `-3514.8 umol m-2 s-1`，中位数 `-3225.2`；其量级远大于 corrected EC 差异且方向相反。该结果不能解释为真实局地净通量，而是明确显示：在复杂地形下，残余 `w_mean` 与背景 CO2 直接相乘会产生不守恒、不可用于局地交换解释的巨大项。

## 当前解释边界

- 本轮已把“主要是 rotation 造成”降为次要解释：rotation 对累计量有约 `17%` 的系统贡献，但核心差异主要来自两塔原始垂直湍流协方差不同。
- `sigma_w` 的条件关联支持两塔湍流状态不对称，但其解释力明显低于 `cov(w,c)`；不能仅凭 `sigma_w` 证明具体地形环流或足迹来源。
- 当前可以将来源推进到“垂直湍流协方差差异主导、rotation 和后续修正次要”，但尚不能在不引入足迹、塔高/下垫面或切面证据时进一步命名具体空间机制。
