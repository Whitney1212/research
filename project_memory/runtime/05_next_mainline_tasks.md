# 下一步主线任务

## 当前总目标

下一阶段的目标不是继续扩大计算种类，而是把现有 raw-w、EA/EC、FL 切面、三站水平风和 CO2 廓线现象收束到可验证的物理判断上：09:00 CO2 次高峰究竟主要来自夜间储存释放、水平平流、横谷向局地环流、湍流交换，还是这些过程的组合。[来源: 用户当前对话 2026-05-21] [推断: 基于当前项目进展整理]

2026-05-26 后，总目标进一步收束为两个优先问题。第一，次高峰基本不再优先按湍流主控解释，而是检查日出后固定时段是否有随风向转变输入的高 CO2 气团，并区分峰后 CO2 下降是本地吸收/稀释还是继续平流带走。第二，raw-w 总输送继续围绕垂直风或垂直空气量主导解释，重点验证 `CVT` 日出后下沉是否对应谷底补偿下沉，以及 `MT/FL` 上升结构是否与坡面热力上坡风和局地次级环流相容。[来源: 用户当前对话 2026-05-26] [已核验: project_memory/evidence/verifications/2026-05-26_two_main_mechanism_directions.md]

## 论文输出方向目标

2026-05-28 后，下一阶段可以围绕一篇复杂地形通量计量小论文组织两个输出方向。第一条方向是把 `CVT` 下沉主导而 `MT/FL` 上升主导写成复杂地形下表观界面输送与地形平均流诊断问题，重点解释 raw \(wc\) 为什么由 `w_mean` 或空气量项主导，并通过风向扇区、`u/v` 残差、FL 切面形态和坐标旋转敏感性检验区分真实地形输送、流线倾斜和仪器坐标投影。第二条方向是把日出后 CO2 先下降再上升的次高峰写成复杂地形晨间转换期事件归因问题，重点比较廓线结构切换、风向转变、三站同步性、FL 空间形态、标准 EC 浓度异常通量和峰后下降速率之间的先后关系。[来源: 用户当前对话 2026-05-28] [已核验: project_memory/evidence/verifications/2026-05-28_paper_direction_targets.md]

这两个方向可以服务于同一篇小论文，但共同边界是：不要把 raw \(wc\) 直接写成传统生态系统 CO2 通量，也不要把量级较小的标准 EC/EA 湍流项写成无意义。更稳妥的论文目标是说明复杂地形中表观空气量输送与 CO2 浓度异常交换发生了解耦，并用现有三站 EC、FL 切面、三站水平风和 CO2 廓线数据建立可复核的机制候选排序。[来源: 用户当前对话 2026-05-28] [推断: 基于当前项目方法边界和论文目标整理]
